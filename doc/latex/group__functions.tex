Smoothing function intended to generate rounded wingtips

Evaluate the function $f(t)=1$, $t<t_{0}$, $f(t)=p(t-t_{0})(1-t)^{\eta}/(1-t_{0})^{\eta}$, $t\geq t_{0}$, where $p(t-t_{0})$ is a polynomial selected to make $f(t)$ $C^{2}$ continuous.


\begin{DoxyParams}{Parameters}
{\em eta} & $\eta$ tip radius exponent; \\
\hline
{\em t0} & $t_{0}$ limit of constant region; \\
\hline
{\em t} & $t$ input variable.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
$f(t)$ on success.
