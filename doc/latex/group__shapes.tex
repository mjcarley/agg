Evaluate NACA four digit aerofoil shape

Evaluate NACA four digit aerofoil shape using formulae of Ladson, Charles L., Brooks, Cuyler W., Jr., Hill, Acquilla S., Sproles, Darrell W., `Computer Program to Obtain Ordinates for NACA Airfoils', NASA-\/\+TM-\/4741, 1996

{\texttt{ https\+://ntrs.\+nasa.\+gov/citations/19970008124}}

Resulting section is NACApmxx, e.\+g. NACA2312 would have $t=0.12$, $p=0.02$, $m=0.3$.

\hline

\hline

\end{DoxyParams}
